## 2. 개요
 - **Eclipse와 AndroidSDK**로 구현한 **오늘의 운세 애플리케이션**입니다. 


 
## 3. 특징
- 사용자의 이름과 생년월일을 받아 난수를 발생시켜 하나의 운세결과가 나오도록 구현하였습니다.
- 안드로이드 스튜디오가 아닌 이클립스로 제작하였습니다.



## 4. 개발 환경 & 개발 툴
- Programing Language : Java 8
- OS: Windows 10
-----------------------------------------------------------------------------
- Tool : Eclipse
- Graphic Tool : Adobe Photoshop CS6
